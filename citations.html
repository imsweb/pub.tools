publication_citation

pub.tools.citations.publication_citation(publication: JournalRecord | BookRecord | ChapterRecord, html: bool = False, use_abstract: bool = False, link: bool = False) → str
Undefined publication type. Only usable with pub types that can be retrieved from Pubmed.

Example usage:

from pub.tools import entrez
from pub.tools import citations
if pub := entrez.get_publication(pmid=12345678):
     citations.publication_citation(publication=pub)
Optional parameters:

html = will render the citation with html tags use_abstract = will include the abstract after the citation (requires html=True) link = will include PubMed link (requires html=True and pmid)
